NETS DNA — Persona Image Assets
================================

Drop persona images into THIS folder. The app will automatically show them
in the persona cards and popup when the file exists. If a file is missing,
a themed placeholder (emoji + icon) is shown instead — so it is safe to add
images one at a time.

FILE NAMING — must match exactly (kebab-case, no spaces):
---------------------------------------------------------
  bubble-tea-royalty.png    →  Bubble Tea Royalty 🧋
  hawker-regular.png        →  Hawker Regular 🍜
  cafe-dweller.png          →  Cafe Dweller ☕
  cinema-hopper.png         →  Cinema Hopper 🎬
  fitness-junkie.png        →  Fitness Junkie 💪
  fashion-forward.png       →  Fashion Forward 🛍️
  tech-enthusiast.png       →  Tech Enthusiast 💻
  foodie-explorer.png       →  Foodie Explorer 🍣
  night-owl.png             →  Night Owl 🦉
  bookworm.png              →  Bookworm 📚
  fast-food-fan.png         →  Fast Food Fan 🍟
  grocery-guru.png          →  Grocery Guru 🛒

IMAGE SPEC:
-----------
  Size:     600 × 600 px (square)
  Format:   PNG or WEBP
  Fit:      object-fit: cover (centre-cropped by the app)
  Style:    Bold, colourful illustrations or stylised portraits work best.
            Keep the main subject centred — edges may be cropped on the card.

Only the 3 personas the user has earned are shown on-screen at any time,
but all 12 files can be pre-loaded so any persona that surfaces looks great.
